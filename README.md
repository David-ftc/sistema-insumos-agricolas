#  Sistema de Gestión de Insumos Agrícolas

##  Descripción
Este proyecto corresponde a la Práctica 3 de Ingeniería de Software, donde se implementa un sistema para gestionar insumos agrícolas utilizando metodologías del SDLC y herramientas de Inteligencia Artificial.

---

## Objetivo
Optimizar los procesos de:
- Control de inventario
- Registro de producción
- Gestión de clientes y proveedores
- Generación de reportes
- Control de acceso por roles

---
